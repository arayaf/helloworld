เรียนรู้ ดิจิทัล
เปลี่ยนวิธีทำงาน

หลักสูตรพัฒนาทักษะ Google Workspace + MS 365 + AI สำหรับเจ้าหน้าที่สนับสนุนและผู้สนใจ

18Sessions
30นาที/session
1–5คน/กลุ่ม
ฟรีไม่มีค่าใช้จ่าย
💡

เคยส่งปฏิทินนัดประชุม แล้วเวลาไม่ตรงกัน? ไม่รู้ว่าไฟล์ควรเก็บใน Drive หรือ OneDrive?
หรือทำงานซ้ำๆ ที่อยากให้ระบบทำแทน? — หลักสูตรนี้แก้ทุกปัญหานั้น
🖥️ ไม่มี Notebook ไม่เป็นไร — สำนักฯ มี Notebook เตรียมไว้ให้

⚡ เมื่อจบหลักสูตร คุณจะทำได้

📅

ส่ง invite ได้ถูกต้อง 100%timezone ตรง, ลิงก์ Meet/Teams ถูกต้อง

🗂️

จัดไฟล์อย่างเป็นระบบไฟล์ Google → Drive | ไฟล์ MS → OneDrive

📊

ใช้ Sheets เป็นฐานข้อมูลสูตร VLOOKUP/IF/COUNTIF + รับข้อมูลจาก Form

⚙️

สร้าง Workflow อัตโนมัติForm → Email แจ้ง → Calendar → Report อัตโนมัติ

🤖

ใช้ Gemini AI ช่วยงานdraft อีเมล, สรุปเอกสาร, สรุปประชุม

💻

เขียน Apps Script ได้automate งานซ้ำๆ ด้วย AI ช่วยเขียนโค้ด

📚 เนื้อหาทั้ง 18 Sessions

Phase 1 — Digital Workspace Literacy (S1–S5b)
S1 · Calendar & Contacts
ปฏิทิน + Meeting Link + Label
  • สร้าง event + timezone ถูกต้อง
  • ลิงก์ Meet/Teams ไม่หลุด
  • Label กลุ่ม invite ในคลิกเดียว
🎯 แก้ปัญหาปฏิทินไม่ตรง 📄
S2 · Gmail & Outlook
อีเมล์มืออาชีพ
  • Signature + CC/BCC ถูกต้อง
  • Schedule send
  • Gemini draft อีเมล์ให้
S3 · Google Drive
จัดเก็บไฟล์ Google
  • โฟลเดอร์เป็นระบบ
  • Shared Drive + permission
  • ไฟล์ Google เท่านั้น ไม่กิน quota
🟢 ไฟล์ Google เท่านั้น
S3b · OneDrive
จัดเก็บไฟล์ Microsoft
  • ซิงก์ .docx/.xlsx/.pptx อัตโนมัติ
  • แชร์ link + กำหนดวันหมดอายุ
  • Version history 30 วัน
🔵 ไฟล์ MS เท่านั้น
S4 · Docs & Word Online
ทำงานร่วมกัน Real-time
  • Co-edit + Comment/Suggest
  • Version history
  • Gemini draft หนังสือ
S5 · Gemini AI ภาพรวม
AI ใน Workspace ทุก app
  • Gemini ใน Gmail/Docs/Sheets/Meet
  • เขียน Prompt ที่ดี
  • ข้อควรระวังข้อมูลลับ
🤖 AI session
Phase 2 — Google Sheets as Database + AI Automation (S6–S12)
S6 · Sheets พื้นฐาน
ตารางข้อมูลที่ถูกต้อง
  • Data Validation (Dropdown)
  • Freeze + Filter
  • ถามสูตรด้วยภาษาไทย
S7 · VLOOKUP/IF/COUNTIF
สูตรที่จำเป็นสำหรับธุรการ
  • ดึงข้อมูลข้ามชีต
  • เงื่อนไขอัตโนมัติ
  • นับตามเงื่อนไขได้
S8 · Sheets เป็น DB 🔑
คิดแบบฐานข้อมูล
  • 1 row = 1 record
  • ไม่ merge cell
  • Gemini ช่วย design ตาราง
🔑 จุดเปลี่ยนสำคัญ
S9 · Forms → Sheets
รับข้อมูลอัตโนมัติ
  • รับใบลา/ลงทะเบียนแทน email
  • ข้อมูลเข้า Sheets เอง
S10 · Workspace Studio
Automation ไม่ต้องเขียนโค้ด
  • Form → Email อัตโนมัติ
  • Sheets → Calendar event
  • Drive → แจ้งเตือน Chat
🤖 AI session
S11–S12 · Dashboard & AI
Looker Studio + NotebookLM
  • Dashboard จาก Sheets
  • ถามคำถามจากเอกสารองค์กร
🤖 AI session
Phase 3 — Google Apps Script + AI (S13–S18)
S13 · Apps Script เบื้องต้น
เปิด Editor ครั้งแรก
  • Script แรก + Trigger
  • Gemini เขียน Script ให้
S14 · อ่าน-เขียน Sheets
Script ควบคุม Sheets
  • getValues / setValue / loop
  • สีแถวอัตโนมัติตามสถานะ
S15 · Email อัตโนมัติ
ส่ง Email จาก Sheets
  • GmailApp.sendEmail()
  • Reminder อัตโนมัติ
S16 · Script ด้วย Gemini AI
AI เขียนโค้ดให้คุณ
  • Prompt Pattern 4 ส่วน
  • Mini-project งานจริง
🤖 AI session
S17 · Gemini in Meet
สรุปประชุมอัตโนมัติ
  • Action items หลังประชุม
  • Transcript อัตโนมัติ
S18 · Capstone 🏆
AI Workflow ครบวงจร
  • Form→Sheets→Auto→Report
  • นำเสนอ workflow ตัวเอง
🏆 Capstone

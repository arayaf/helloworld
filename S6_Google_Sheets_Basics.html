🎓 Phase 1 · S6

Google Sheets พื้นฐาน — มากกว่าตาราง Excel

📅 จันทร์ 13 ก.ค. 2569 ⏱ 30 นาที 🕓 16:00–16:30 น. 📍 ตึกอธิการบดี
🎯 วัตถุประสงค์
ใช้ Google Sheets สร้างตารางข้อมูลที่ถูกต้อง กรองข้อมูลได้ และเป็นพื้นฐานสำหรับ automation ใน Phase 2
✅ ผลลัพธ์การเรียนรู้
  • Format ข้อมูลให้เป็น Table ที่ถูกต้อง: 1 row = 1 record, ไม่ merge cell
  • ใช้ Data Validation สร้าง Dropdown list จำกัดค่าที่กรอกได้
  • ใช้ Freeze ล็อก header และ Filter กรองข้อมูลตามเงื่อนไข
  • ถามสูตรเป็นภาษาไทยด้วย Gemini in Sheets ได้
⏱ แผนการสอน 30 นาที
เวลาหัวข้อรายละเอียด
0–5 นาที Table ที่ถูกต้อง Header row ชั้นเดียว / ไม่ merge cell / ไม่แถวว่างกลางข้อมูล — เปรียบเทียบ 'ก่อน vs หลัง'
5–12 นาที Data Validation เลือกคอลัมน์ → Data → Data validation → List of items → ใส่ค่า Dropdown
12–18 นาที Freeze & Filter View → Freeze → 1 row / Data → Create a filter → กรองตามค่าหรือเงื่อนไข
18–25 นาที ฝึกปฏิบัติ สร้าง Sheet ตัวอย่างจากงานจริง — ใส่ Dropdown, Freeze header, Filter ตาม status
25–28 นาที Gemini in Sheets คลิก Gemini icon → พิมพ์ภาษาไทย: 'สร้างสูตรนับแถวที่สถานะเป็น รอดำเนินการ'
28–30 นาที สรุป Table ที่ดี = ฐานของ automation ทั้งหมดใน Phase 2 และ 3
🧰 สิ่งที่ต้องเตรียม
  • Google Sheets เปล่าสำหรับฝึก
  • ข้อมูลตัวอย่างจากงานจริง เช่น รายการติดตามงาน หรือบัญชีวัสดุ (เตรียมล่วงหน้า)
🤖 Gemini in Sheets: คลิก Gemini icon (มุมขวาบน) → พิมพ์เป็นภาษาไทย เช่น 'สร้าง dropdown list สำหรับ สถานะงาน ให้มีตัวเลือก รอดำเนินการ กำลังดำเนินการ เสร็จแล้ว' → Gemini สร้างให้อัตโนมัติ
📌 หมายเหตุผู้สอน: S6 เป็นจุดเริ่มต้น Phase 2 — ถ้าผู้เรียนยังไม่เข้าใจ 'Table ที่ถูกต้อง' ควรใช้เวลากับส่วนนี้มากขึ้น ก่อนไปเรื่องสูตรใน S7 เพราะ S8 (Sheets as DB) จะสร้างต่อจากพื้นฐานนี้โดยตรง

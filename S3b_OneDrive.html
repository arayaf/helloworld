OneDrive — จัดเก็บไฟล์ Microsoft

📅 พุธ 8 ก.ค. 2569 ⏱ 30 นาที 🕓 16:00–16:30 น. 📍 ตึกอธิการบดี
🎯 วัตถุประสงค์
จัดการไฟล์ MS Office ผ่าน OneDrive ให้เป็นระบบ ซิงก์อัตโนมัติ และแชร์ถูกต้องโดยไม่กิน Google quota
✅ ผลลัพธ์การเรียนรู้
  • ติดตั้ง OneDrive desktop app และซิงก์ไฟล์อัตโนมัติ
  • แชร์ไฟล์ .docx/.xlsx/.pptx ผ่าน OneDrive link พร้อมตั้ง permission
  • กำหนดวันหมดอายุลิงก์แชร์
  • ใช้ Version history กู้คืนไฟล์เก่าภายใน 30 วัน
  • รู้กฎ: ไฟล์ MS ทั้งหมด → OneDrive เท่านั้น
⏱ แผนการสอน 30 นาที
เวลาหัวข้อรายละเอียด
0–5 นาที ทำไมต้อง OneDrive เปรียบเทียบ: .docx ใน Google Drive vs OneDrive — quota / format / version history
5–12 นาที ติดตั้ง & ซิงก์ ดาวน์โหลด OneDrive app → Sign in บัญชี Microsoft → เลือกโฟลเดอร์ที่จะซิงก์
12–20 นาที แชร์ไฟล์ คลิกขวา → Share → Copy link → Can view / Can edit → Set expiry date
20–25 นาที Version history คลิกขวาที่ไฟล์ → Version History → เลือก Restore — กู้งานที่หายได้ภายใน 30 วัน
25–28 นาที Copilot in OneDrive เลือกไฟล์ → Summarize / Ask questions about the file (MS 365 Copilot)
28–30 นาที สรุปกฎ ไฟล์ MS ทุกชนิด (.docx .xlsx .pptx .pdf) → OneDrive เท่านั้น — ไม่ลาก drop ใน Google Drive
🧰 สิ่งที่ต้องเตรียม
  • Windows หรือ Mac ที่ติดตั้ง OneDrive app ได้
  • บัญชี Microsoft 365 ที่ login แล้ว
  • ไฟล์ .docx และ .xlsx ตัวอย่างสำหรับฝึกแชร์และดู version history
🤖 Copilot in OneDrive: เลือกไฟล์ Word หรือ Excel → คลิก Summarize this file → ได้สรุปเนื้อหาโดยไม่ต้องเปิดไฟล์ — ใช้ได้กับ MS 365 ที่มี Copilot
📌 หมายเหตุผู้สอน: จุดสำคัญที่ผู้เรียนมักสับสน: OneDrive กับ Google Drive ใช้แทนกันได้ไหม? — คำตอบคือไม่ได้ เพราะ quota และ format ต่างกัน ต้องอธิบายให้ชัดด้วยตัวอย่างที่จับต้องได้

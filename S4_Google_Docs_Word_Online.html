Google Docs & Word Online — ทำงานร่วมกัน Real-time

📅 พฤหัส 9 ก.ค. 2569 ⏱ 30 นาที 🕓 16:00–16:30 น. 📍 ตึกอธิการบดี
🎯 วัตถุประสงค์
ทำงานร่วมกันบนเอกสารแบบ real-time ได้ โดยไม่ต้องส่งไฟล์ไปมา
✅ ผลลัพธ์การเรียนรู้
  • แก้ไขเอกสารร่วมกันหลายคนพร้อมกันได้ใน Google Docs และ Word Online
  • ใช้ Comment mode เพื่อแสดงความคิดเห็นโดยไม่แก้ต้นฉบับ
  • ใช้ Suggesting mode เพื่อเสนอแก้ไขที่เจ้าของเอกสาร Accept/Reject ได้
  • ดูและกู้คืน Version history
  • ใช้ Gemini draft หนังสือราชการและปรับ tone ของเอกสาร
⏱ แผนการสอน 30 นาที
เวลาหัวข้อรายละเอียด
0–5 นาที Co-editing Demo เปิด Docs พร้อมกัน 2 หน้าจอ — แสดง cursor ของกันและกัน ชี้ให้เห็นการเปลี่ยนแปลงแบบ real-time
5–12 นาที Comment mode เลือกข้อความ → Insert Comment → ตอบ Comment → Resolve — ใช้แทนการส่ง email แก้ไข
12–18 นาที Suggesting mode View → Suggesting → แก้ไขโดยไม่ทำลายต้นฉบับ → เจ้าของ Accept หรือ Reject
18–24 นาที Version history File → Version history → See version history → ดูการเปลี่ยนแปลง → Restore version
24–28 นาที Gemini in Docs Help me write → draft หนังสือ / Tone → More formal / สรุปเอกสารยาว
28–30 นาที สรุป Comment vs Suggest / Version history / Gemini draft — ส่ง link แทนส่งไฟล์เสมอ
🧰 สิ่งที่ต้องเตรียม
  • Google Doc ที่แชร์กับผู้เรียนทุกคนล่วงหน้า (เตรียมก่อนสอน)
  • หน้าจอ 2 จอสำหรับ demo co-editing หรืออุปกรณ์ 2 เครื่อง
  • เอกสารตัวอย่างที่มีเนื้อหาพอสมควร (ไม่สั้นเกินไป)
🤖 Gemini in Docs: Help me write → บอกว่าต้องการเขียนอะไร → Gemini draft ให้ / จากนั้นคลิก Tone → More formal เพื่อปรับภาษาให้เป็นทางการ — ใช้กับหนังสือราชการได้ดีมาก
📌 หมายเหตุผู้สอน: เตรียม Google Doc ที่แชร์กับผู้เรียนทุกคนไว้ล่วงหน้า เพื่อให้ทดลอง co-edit ได้ทันที อย่าให้ผู้เรียนเสียเวลาสร้าง Doc เองในห้อง
